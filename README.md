# Нейроморфный ПИД-регулятор для двигателя постоянного тока на платформе Kaspersky Neuromorphic Platform

## Описание проекта

Данный проект посвящён реализации ПИД-регулятора на нейроморфном процессоре с использованием **Kaspersky Neuromorphic Platform (KNP)**.

В качестве объекта управления используется **двигатель постоянного тока (ДПТ)**. Для получения обучающей выборки применяется классический ПИД-регулятор, который выступает в роли «учителя». Нейронная сеть обучается воспроизводить его управляющее воздействие, после чего спайковая нейронная сеть размещается на нейроморфной архитектуре и используется для управления двигателем.

В проекте проводится сравнение:

* классического ПИД-регулятора;
* нейроморфного регулятора на основе спайковой нейронной сети (SNN).

Оценка качества регулирования выполняется как при отсутствии шума измерений, так и при наличии шума.

# Математическая модель двигателя

В качестве объекта управления используется двигатель постоянного тока независимого возбуждения.

## Электрическая часть

Ток якоря определяется выражением:

$$
I_a = \frac{U - c_e \Phi \omega}{R}
$$

где:

* $U$ — напряжение якоря;
* $I_a$ — ток якоря;
* $c_e$ — коэффициент противо-ЭДС;
* $\Phi$ — магнитный поток;
* $R$ — суммарное сопротивление цепи якоря.

## Механическая часть

Динамика угловой скорости описывается уравнением:

$$
J\frac{d\omega}{dt}=c_m\Phi I_a-B\omega-M_{load}
$$

где:

* $J$ — момент инерции ротора;
* $c_m$ — коэффициент электромагнитного момента;
* $B$ — коэффициент вязкого трения;
* $M_{load}$ — момент нагрузки;
* $\omega$ — угловая скорость.

# Архитектура нейронной сети

Нейроморфный регулятор состоит из трёх блоков:

```text
Входные данные
      │
      ▼
+----------------+
|    Encoder     |
| Dense+Sigmoid  |
+----------------+
      │
      ▼
+----------------+
|      SNN       |
| TernaryDense   |
| Heaviside      |
+----------------+
      │
      ▼
+----------------+
|    Decoder     |
| Dense Linear   |
+----------------+
      │
      ▼
Управляющее воздействие
```

## Входные параметры

На вход сети подаются:

* ошибка регулирования;
* производная ошибки;
* интеграл ошибки;
* текущая скорость двигателя;
* заданная скорость.

Вектор входных данных:

```text
[e, de/dt, ∫e·dt, ω, r]
```

# Обучение нейронной сети

## ПИД-регулятор-учитель

Для генерации обучающей выборки используется ПИД-регулятор:

$$
u = K_p e + K_i \int e,dt + K_d \frac{de}{dt}
$$

Параметры регулятора:

| Параметр | Значение |
| -------- | -------- |
| Kp       | 2.0      |
| Ki       | 0.5      |
| Kd       | 0.01     |

---

## Генерация обучающих данных

Обучающая выборка формируется путём моделирования работы ДПТ при случайно изменяющихся заданиях скорости.

Размер выборки:

```text
75000 примеров
```

Каждый пример содержит:

Вход:

```text
[e, de/dt, integral, omega, reference]
```

Выход:

```text
u_PID
```

## Конфигурация сети

### Кодировщик

```python

```

### Спайковый слой

```python

```

### Декодер


# Метрики качества

Для оценки качества регулирования используются следующие показатели.

## Перерегулирование

Максимальное превышение установившегося значения.

## Время нарастания

Время перехода от 10 % до 90 % установившегося значения.

## Время установления

Время входа сигнала в полосу ±5 % относительно установившегося значения.

## RMSE

Среднеквадратическая ошибка отслеживания задания.
